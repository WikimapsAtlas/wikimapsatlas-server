Query the world!

Introduction

The Wikimaps Atlas project is an open source wiki geo data engine to create data driven maps for the web.

Hierarchical Administrative Subdivision Codes

HASC Codes are used as the primary identifier to refer to a geographical area, rather than an x/y/z tile system

Methods

  • /world
     List countries of the world
  • /world/HASC
     List administrative subunits within country
  • /bbox/HASC
     Return bounding box of the area
  • /center/HASC
     Return centroid of the area
  • /geojson/HASC
     
    /topojson/HASC
     Return geojson or topojson data of area

Data

  • Administrative data is from Natural Earth with custom joins from the Wikimaps Atlas project (Public Domain)
